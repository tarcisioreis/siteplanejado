A) A Quick Start Guide - top

If you already have a mobile website or a website with responsive layout, it takes only a several minutes to create your own WebView Application. All you need to do is to change the name of the Application and change the link of homepage of the Application to your website's homepage.

We created a video guide for quick start. You can watch it here.

For this, first of all, open the project source with Eclipse (ADT plugin must be installed to create .apk file of the Application).

Now open 'res/values/strings.xml' file. This file contains all string values, which are used in Application forms.

In this file, you need to modify first 2 strings.
- 'app_name' is your application's name
- 'base_url' is a link to main web page

Congratulations! You've just finished creating your own WebView Application.

 

Further modifications:

Localization
To localize the application to your preferred language, modify strings that start with 'action_'.

Splash Screen
Replace 'logo.png' and 's_img.png' in 'res/drawable-hdpi' folder with your own images.

Page loading
Replace 'load_bg.png' in 'res/drawable-hdpi' folder to change background image with your own logo or preferred image.

'No connection' error page
Replace 'logo_er.png' in 'assets' folder with your own logo.

AdMob
Edit 'admob_publisher_id' in 'res/values/strings.xml' with your own publisher ID.

Note: Rate action works only after uploading your application to Google Play market. Otherwise, you'll get an error.

B) Application File structure - top

MainActivity.java is a main Application activity file
Splashscreen.java opens a splashscreen on Application start.
Image.java helps to open image with fullscreen and zoom control

This folder contain error page files. When there is no internet connection, this page will be shown. Just replace 'logo_er.png' with your own logo.

C) Web page usage Tips - top

Application behaves as a regular web browser (like Chrome, etc).

The only thing you should be aware of is:
- To create a link for One Touch Call action, 'href' property of the link must have the format of 'tel:yourphonenumber'.
Ex.: <a href="tel:777777777">Call</a>.

- To create a link for One Touch SMS Send action, 'href' property of the link must have the format of 'sms:yourphonenumber'.
Ex.: <a href="sms:777777777">Send SMS</a>.

- To create a link for One Touch Share action, 'href' property of the link must have the format of 'share:link/url/text'.
Ex.: <a href="share:yourlink.com">Share</a>.

- To create a link for MP3, MP4, 3gp open action, 'href' property of the link must have the format of 'yoururl/name.mp3', 'yoururl/name.mp4', 'yoururl/name.3gp'. End of File Extension
Ex.: <a href="http://mosaicdesign.uz/media/example.mp3">MP3</a>.

- To create a link for .pdf, .txt, .ttf, xls, .rar open file action, 'href' property of the link must have the format of 'yoururl/name.txt', 'yoururl/name.pdf', 'yoururl/name.ttf'. End of File Extension and opened by Google Viewer service
Remember:More servers don't allow to upload .rar or .xls extensions. Need to change your server config file or turn on this services
Ex.: <a href="http://mosaicdesign.uz/file/example.pdf">Example.pdf</a>.
- To create a link for .mp3, .mp4, .pdf download file action, 'href' property of the link must have the format of 'download:http://yoururl/name.mp3', 'download:http://yoururl/name.mp4'
Ex.: <a href="download:http://mosaicdesign.uz/file/example.pdf">Download.pdf</a>.
